# Website Security Hardening Project

This repository documents the remediation, security hardening, and performance optimization of a WordPress environment hosted on a Linux cloud server.

## Scope of Work
- Incident investigation and malware remediation
- WordPress core and plugin cleanup
- Cloudflare security configuration and cache optimization
- Server access restructuring and security improvements
- Documentation of repeatable remediation steps

## Tools & Technologies
- Linux (SSH)
- WordPress
- Cloudflare
- Wordfence
- Basic server security practices

## Outcome
Restored website functionality, improved security posture, and implemented monitoring for ongoing protection.
